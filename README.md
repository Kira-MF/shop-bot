# Shop Bot

Telegram магазин бот для продажи цифровых товаров.

## Возможности

- Каталог с категориями и товарами
- Система баланса (пополнение через скриншот)
- Подтверждение/отклонение оплаты админом
- Выдача логин/паролей после покупки
- История заказов
- Поддержка SOCKS5 прокси
- Админ панель:
  - Добавить/удалить категорию и товар
  - Изменить реквизиты
  - Изменить баланс пользователю
  - Список пользователей и статистика

## Установка

```powershell
py -m pip install -r requirements.txt
```

## Настройка

Создай `.env` рядом с `shop_bot.py`:

```env
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_ID=твой_Telegram_ID
PROXY=
```

Если прокси не нужен, оставь `PROXY` пустым.

## Запуск

```powershell
py shop_bot.py
```

## База данных

Хранится в `shop.json` - создаётся автоматически при первом запуске.
